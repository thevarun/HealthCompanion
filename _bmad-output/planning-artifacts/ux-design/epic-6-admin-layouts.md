### Changes from Tablet

- **Sidebar**: Hidden, accessible via hamburger menu (Sheet component)

- **Metric Grid**: Single column stack
- **Page Padding**: Reduced to 16px
- **Tables**: Full horizontal scroll, min-width enforced
