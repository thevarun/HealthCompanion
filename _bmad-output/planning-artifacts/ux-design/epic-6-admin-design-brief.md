# Admin Panel Design Brief

## Overview

A comprehensive admin panel for system management, providing administrators with tools to manage users, view feedback, monitor system activity, and test email templates. The design follows a clean, professional aesthetic inspired by Linear, Vercel, and Stripe dashboards.

## Design Decisions

### Visual Style
- **Theme**: Light mode default with dark mode toggle
- **Color Palette**: High contrast - dark sidebar on light theme, pure black sidebar on dark theme
- **Typography**: Inter font family for clean, modern appearance
- **Icons**: Lucide icons for consistency with shadcn/ui

### Layout Architecture
- **Collapsible Sidebar**: Full sidebar (240px) with icon-only collapsed mode (64px)
- **Responsive Breakpoints**: Desktop (1024px+), Tablet (768-1023px), Mobile (<768px)
- **Mobile Navigation**: Sheet/overlay pattern for sidebar on mobile
- **Content Area**: Max-width containers with consistent padding

### Interaction Patterns
- **Micro-animations**: 200ms transitions for UI feedback
- **Card animations**: Staggered entrance on page load
- **Hover states**: Subtle background changes and shadows
- **Loading states**: Skeleton loaders for data tables

### Data Display
- **Tables**: Sortable columns, row selection with checkboxes, pagination
- **Status indicators**: Color-coded badges (Active=green, Suspended=red, Pending=yellow)
- **Metrics**: Card-based display with trend indicators

## Visual Direction

### Design Language
- Clean, minimal interface with focus on content
- Generous whitespace for scannability
- Consistent 8px spacing grid
- Rounded corners (8px for cards, 6px for buttons)
- Subtle shadows for depth (not heavy drop shadows)
