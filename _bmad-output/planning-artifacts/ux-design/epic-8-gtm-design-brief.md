# Epic 8: Go-To-Market Features — Design Brief

## Overview

Three Go-To-Market components that help users share content (8.1), generate private shareable URLs (8.2), and drive organic traffic via programmatic SEO pages (8.5). All components follow the project's dark-mode-first design system with semantic color tokens, shadcn/ui primitives, and lucide-react icons.

## Design Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Animation library | CSS transitions (Tailwind `transition-*`) | Project does not use framer-motion; keeps bundle lean |
| Icon system | lucide-react | Already a project dependency; consistent with existing components |
| Color tokens | Semantic (`bg-muted`, `text-foreground`, `bg-primary`) | Ensures dark/light mode consistency without hardcoded colors |
| Copy feedback | sonner toast | Already configured in the project; non-blocking |
| Share popup | shadcn Popover | Accessible, keyboard-navigable, consistent with project primitives |
| Share link modal | shadcn Dialog | Already installed; handles focus trap and overlay |
| pSEO page nav/footer | Project layout (standalone for public pages) | pSEO pages are unauthenticated, may use a stripped-down layout |
| pSEO FAQ | shadcn Accordion | Already installed; accessible and keyboard-friendly |
| Class merging | `cn()` from `@/utils/Helpers` | Project-standard utility |

## Visual Direction

### Color System
- **Neutral surfaces:** `bg-muted`, `bg-card`, `bg-background` (not `neutral-*` or `slate-*`)
- **Text:** `text-foreground`, `text-muted-foreground` (not `text-slate-*`)
- **Borders:** `border-border` (not `border-slate-200`)
- **Primary accent:** `bg-primary`, `text-primary`, `text-primary-foreground`
- **Platform-specific hover colors** (ShareWidget only): Keep branded colors (`hover:bg-[#0A66C2]` for LinkedIn, `hover:bg-[#1877F2]` for Facebook) — these are external brand colors, not part of the design system
- **Status badges** (Shareable Links): Use project's `badge` component variants

### Typography
- System font stack via Tailwind defaults
- Heading sizes: `text-3xl`/`text-4xl` for page headings, `text-xl`/`text-2xl` for section headings
- Body: `text-sm`/`text-base`

### Spacing & Layout
- Consistent with project patterns: `p-6` sections, `gap-4` grids
- Mobile-first responsive breakpoints (`sm:`, `md:`, `lg:`)
- Touch-friendly targets: `min-h-[44px]` on interactive elements

### Animations
- `transition-colors duration-200` for hover states
- `transition-opacity duration-150` for copy feedback icon swap
- No framer-motion anywhere — purely CSS/Tailwind transitions

## Inspiration Sources

- **ShareWidget:** Twitter/LinkedIn native share dialogs, Web Share API spec
- **Shareable Links:** Notion share modal, Google Docs share dialog, Linear project sharing
- **pSEO Template:** Zapier's programmatic pages (`/apps/*`), G2 category pages, NerdWallet comparison pages

## Cross-Cutting Patterns

All components across Epic 8 follow these rules:

1. **`'use client'`** directive on all interactive components
2. **Barrel exports** via `index.ts` per feature folder (`src/components/share/index.ts`, `src/components/pseo/index.ts`)
3. **Analytics hook points** as `// TODO: Analytics — event: "event_name", properties: { ... }` comments (wired in Epic 9)
4. **No framer-motion** — use Tailwind `transition-*` classes
5. **Semantic tokens** — no hardcoded color values except external brand colors
6. **lucide-react** icons — no inline SVGs
7. **`cn()`** from `@/utils/Helpers` for className merging
8. **sonner toast** for user feedback (copy confirmation, link created, etc.)
